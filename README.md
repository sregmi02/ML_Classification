# ML_Classification
There are 3 files:
SubratRegmi_DecisionTree contains codes for applying Decision Tree Algorithm on the iris dataset to determine the type of iris flower. 
SubratRegmi_KNN_DecisionBoundary contains codes for applying KNN algorithm on the iris dataset and to plot the decision boundaries for different values of k. 
SubratRegmi_KNNPerformanceEvaluation contains codes for applying KNN algorithm on the iris dataset and evaluate its performance using Precision, Recall, F-1Score, and AUC-ROC Curve.
